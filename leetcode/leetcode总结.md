### 接雨水

```
遍历两边，取一个位置左边的最大值保存在dp，取一个位置右边的最大值保存在dp，然后取各个位置两个dp的最小值，减去该位置的高度就是该位置能够的储水量。

用栈来解题也是一个好方法。
```
