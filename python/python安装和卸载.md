* python 3.7安装

```bash
sudo apt-get autoremove python
sudo apt-get autoremove python3
sudo apt-get autoremove python-pip
## 更新源科大或者清华源
sudo apt-get install python3.7
sudo apt-get install python3-pip
ln -s /usr/bin/python3.7 /usr/bin/python
python -m pip install XXX --user

## Python.h找不到
sudo apt-get install libpython3.7-dev && sudo apt-get install python-dev
```
